# Project Memory

This directory is the durable memory layer for the SaltyRN research discussion.

- `PROJECT_STATE.md` is the canonical, editable snapshot of current conclusions.
- `DECISIONS.md` is the compact decision ledger; superseded entries remain visible.
- `PLAN.md` is the live implementation plan with acceptance gates and next actions.
- `DISCUSSION_LOG.md` is chronological and append-only.
- Root `AGENTS.md` defines the read/update protocol for future agents.

Detailed topic notes can remain under `notes/`; the current-state file should link
to them rather than duplicate long analyses.
